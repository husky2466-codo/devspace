# M0 Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the Electron + React project with the full token system, frameless macOS window, and an empty but fully themeable IDE shell (native title bar + app top bar + empty center + status bar).

**Architecture:** Electron main process handles the frameless window and macOS menu bar. The React renderer owns all UI. The IDE shell (`IDE.jsx`) is the single root that holds all layout state and renders the chrome components. No routing library — a single full-screen view. Vite handles the React build; `electron-builder` will handle packaging later.

**Tech Stack:** Electron 30+, React 18, Vite 5, `@vitejs/plugin-react`, CSS custom properties (no CSS-in-JS), Geist + JetBrains Mono via Google Fonts or local woff2. No testing framework needed for M0 — purely visual verification against the prototype.

```jsx
export default function ResizeHandle({ side = 'right', onResize, min = 180, max = 900 }) {
  const onMouseDown = (e) => {
    e.preventDefault();
    const startX = e.clientX;
    const startW = e.currentTarget.parentElement.offsetWidth;

    const move = (ev) => {
      const dx = ev.clientX - startX;
      const raw = side === 'right' ? startW + dx : startW - dx;
      onResize(Math.max(min, Math.min(max, raw)));
    };

    const up = () => {
      document.removeEventListener('mousemove', move);
      document.removeEventListener('mouseup', up);
      document.body.style.cursor = '';
      document.body.style.userSelect = '';
    };

    document.addEventListener('mousemove', move);
    document.addEventListener('mouseup', up);
    document.body.style.cursor = 'col-resize';
    document.body.style.userSelect = 'none';
  };

  return (
    <div
      onMouseDown={onMouseDown}
      style={{
        position: 'absolute',
        top: 0,
        bottom: 0,
        [side === 'right' ? 'right' : 'left']: -3,
        width: 6,
        zIndex: 20,
        cursor: 'col-resize',
      }}
      onMouseEnter={(e) => (e.currentTarget.style.background = 'var(--accent-soft)')}
      onMouseLeave={(e) => (e.currentTarget.style.background = 'transparent')}
    />
  );
}
```

import { useState, useEffect } from 'react';
import NativeTitleBar from './components/NativeTitleBar.jsx';
import TopBar from './components/TopBar.jsx';

  { id: 'forge',     name: 'forge',     activity: 'run',  dirty: true },
  { id: 'archivist', name: 'archivist', activity: 'idle', dirty: false },

export default function IDE() {
  const [themeId, setThemeId] = useState(() => {
    return localStorage.getItem('ds.v3.tone') || 'terminal';

        modified={activeTab.dirty ? 2 : 0}

            position: 'relative',

            width: 32, flexShrink: 0,
